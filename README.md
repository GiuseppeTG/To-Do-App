## Getting Started

To get a local copy up and running, follow these simple example steps.

### Prerequisites
code editor and browser

### Install
`npm i`

### Usage
Type something in the input to add to the to do list. Navbar is just decorative :P

### Run tests
`npm test`

### Deployment
`npm start`
